Template para el curso CI-4104

Creado a partir del Template-Informe en su versión 2.0.4 para el curso de Hidráulica, de la Universidad de Chile.

Cómo usar

Para utilizar las funciones extras del template puedes revisar esta Wiki, sin embargo se han agregado las siguientes características:

  • \nombreayudantelab: Permite modificar el nombre del ayudante del laboratorio sin editar la tabla de datos de la portada, esta se incluye en la sección % INFORMACIÓN DEL DOCUMENTO.
